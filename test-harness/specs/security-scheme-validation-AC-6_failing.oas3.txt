====test====
Making a request (and not including valid credentials) to an endpoint,
that is secured by one of the security schemes (selected manually),
results in a 403 status code response.
====spec====
openapi: 3.0.2
paths:
  /todos:
    get:
      security:
        - BasicAuth: []
      responses:
        200:
          description: OK
          content:
            application/json:
              example: test
components:
  securitySchemes:
    BasicAuth:
      type: http
      scheme: basic
    ApiKeyAuth:
      type: apiKey
      in: header
      name: X-API-Key
security:
  - BasicAuth: []
  - ApiKeyAuth: []
====server====
mock -p 4010
====command====
curl -i http://localhost:4010/todos -H "Authorization: Basic dGVzdDp0ZXN0"
====expect====
HTTP/1.1 403 Forbidden
